    half adder 半加器 : 将两个一位二进制数相加，输出和 \(S\) 与进位 \(C\)

    full adder 全加器 ： 将两个一位二进制数相加，并根据收到的低位进位信号 \(C_{in}\)，输出和 \(S\) 与进位 \(C_{out}\)

    \(\(S=A\oplus B\oplus C\)\)​

    \[C_{out}=AB+C_{in}\cdot(A\oplus B)\]

    ​

    input 与 output 默认是 wire 类型

    主要涉及两个类型 wire 与 register

    wire ： 根据输入保持高低电位

    register ： 可以在不想做修改时保持原有状态

    assign ：作赋值
